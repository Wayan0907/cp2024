1b

第二週起請按照下列座位表入座:

請利用"學號@nfu.edu.tw" 登入 Office 365 後, 利用 Teams 團隊代碼: fqpabi8 進入線上課程

透過新的團隊中"一般"頻道中連結取得個人的作業倉儲, 統一位於 https://github.com/mdecp2024 中的 Repositories

接下來學習如何利用 Github Codespaces: https://github.com/codespaces 直接在線上維護作業倉儲, 唯一的限制是每月免費帳號只能使用 120 core hours, 因此使用最小的虛擬主機 2 cores, 每月限制只能使用 60 小時. 因此沒有進行操作時要關閉作業倉儲對應的 Codespaces 虛擬主機.

新增一個 Codespaces 後, 必須先安裝可以執行啟動作業倉儲編輯的環境, 也就是安裝需要的 Python 模組:

pip install flask flask_cors bs4 lxml pelican markdown gevent
模組安裝結束後, 以 python main.py 啟動編輯

也可以利用可攜程式系統在近端電腦上進行作業倉儲維護 (沒有時間限制, 但要設定維護倉儲的權限):

  1. 啟動可以完全依賴隨身碟中的檔案
  2. 也可以設法將可攜程式下載到 c:\ (固態硬碟) 後, 利用 Hybrid (複合) 方式啟動
